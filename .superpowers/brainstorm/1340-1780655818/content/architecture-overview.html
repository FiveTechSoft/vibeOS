Componentes Clave

🎨 CSS Theme

Librería CSS Windows XP (Luna). Botones, ventanas, menús, tabs, tree-view, campos formulario, status bar, diálogos.

⚡ JS Runtime

Comportamientos pre-cargados: menú dropdowns, tab switching, tree expand/collapse, drag de ventanas, envío de interacciones al backend.

🔧 Backend (Proxy)

Mantiene system prompt + windowSessions. Envía contexto al LLM. Recibe deltas HTML y los reenvía al frontend.

🧠 LLM Engine

Genera HTML inicial y responde a interacciones con deltas. Mantiene coherencia del "SO" + improv/comedia.
